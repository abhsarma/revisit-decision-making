# Tutorial

On this page, you will get to know the graph you will be using to make decisions. Each graph depicts a temperature forecast. Note that the temperature is shown on the Celsius (°C) scale. If you are more familiar with the Fahrenheit (°F) scale, please recall that 0°C equals 32°F, and a change of 1°C is equivalent to a change of 1.8°F.

On the next four pages, you will be asked to perform training tasks. Please read this tutorial carefully before proceeding to the rest of the study.
